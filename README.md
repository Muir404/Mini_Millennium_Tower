# 千年微塔-Mini Millennium Tower
## 项目介绍
这是一个C++&Lua项目，用于模拟一个简单的塔防游戏。全面使用ECS架构与context上下文逻辑，实现游戏的实体-组件-系统架构。

## 外部库
- [Lua](https://www.lua.org/)
- [entt](https://entt.dev/)
- [SDL3](https://www.libsdl.org/)
- [SDL3_image](https://www.libsdl.org/projects/SDL_image_image/)
- [SDL3_ttf](https://www.libsdl.org/projects/SDL_ttf/)
- [SDL3_mixer](https://www.libsdl.org/projects/SDL_mixer/)
- [sol2](https://github.com/sol2/sol2)
- [glm](https://glm.g-truc.net/)
- [nlohmann/json](https://github.com/nlohmann/json)
- [spdlog](https://github.com/gabime/spdlog)
- [imgui](https://github.com/ocornut/imgui)

## 职责说明
- C++：负责游戏的逻辑、物理、渲染等核心功能。
- Lua：负责游戏的脚本、事件处理等。
